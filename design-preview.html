Claude Bridge — v3 design preview

The visual source of truth for the v3 pass. Every token, component, flow, and open decision in PLAN.md is rendered here so you can approve before any production code changes. Reply "ship it" to accept defaults, or override like D6: C, D12: A.

Goal

Sub-50 ms state renders. ~10 ms status line. Motion that reads as intentional, not decorative.

Scope

Visual + brand refresh across webview, terminal, status bar, notifications, filenames, marketplace. No architecture rewrite.

Anti-goal

No new features. No competitive rename. Everything new here is polish on the wedge.
